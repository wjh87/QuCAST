The experimental data files for QuCAST: A Time Series Forecasting Model with Quantum Feature Extraction Designed by Collaborative Quantum Architecture Search are four in total, namely: CSG-SPD, CSG-WPD, HYDRO, and JEDOW. In addition, it contains the experimental code, the eight representative VQC structures used for EQAS, as well as more detailed experimental results for EQAS and QSeqDAS.
